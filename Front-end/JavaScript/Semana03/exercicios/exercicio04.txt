-------------------------------------


var = seria um valor global, ou seja, consegue usar uma variavel de uma função, fora dela, por exemplo
let = variavel de escopo de bloco, uma variavel de uma função só pode ser usada na função, por exemplo
const = um let, porém não é possiível alterar o valor



funcoes = retorna algo
metodos = não retorna nada


Arrays = Uma lista
Objetos = Um lugar que é possível guardar valores com base em uma chave
